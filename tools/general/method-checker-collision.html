1. Собрать сводную модель в Navisworks.
  2. Создать ПН для проверяемых элементов (удобным для Вас способом).
  3. Распределить ПН по группам согласно матрицы коллизий (из EIR).

Для этого создать Excel

 

  1. Подгрузить группы наборов в настройки

 

  1. Открыть инструмент Cheker-добавить проверку 5

 

  1. Создать Excel с правилами проверки (адаптировать матрицу коллизий из EIR)
  1. Импортировать матрицу коллизий в Checker
  1. Запустить проверку.
  1. По результатам проверки выгрузить Excel
  1. Для просмотра коллизий есть возможность переноса проверки в Clash Detective.

 

 

Для этого не закрывая окно Checker, откройте окно Clash Detective. Проверки перенесутся.
