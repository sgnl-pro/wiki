**[2026.13](/general/updates/2026-13)** ^20.07.2026^
  
:sparkles: Добавили галочку `Уникальное значение в рамках проекта` для всех типов полей. При активации пользователи не смогут сохранить форму со значением данного поля, если это значение уже используется в рамках данного типа формы проекта (например для ввода номера документа, ИНН компании, выборе записи ОЖР в АОСР).

:sparkles: Добавили возможность отображать значение поля типа "Список вариантов" в имени формы (например для отображения типа и единиц измерения при выборе материала или вида работ).

:sparkles: Сняли ограничение с количества вариантов у поля типа "Список вариантов": теперь можно оставить один вариант (например для обязательного подтверждения согласия в некоторых формах).

:sparkles: Разрешили использовать цифры в системных именах полей (системные имена используются в шаблонах для [генерации DOCX](/forms/generate)).
