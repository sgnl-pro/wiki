Пользователи с соответствующими правами создают и заполняют экземпляры форм (собственно формы). Форма всегда привязана к конкретному проекту и типу формы.

## Поиск и просмотр форм
  
Выберите тип формы из доступных в выпадающем списке. Для каждого типа формы в проекте доступен реестр (таблица) со списком созданных форм.
  
Колонки таблицы – это поля, которые специальным образом отмечены в конструкторе, а также системные свойства формы: номер, тип в проекте, автор, последний редактор, дата создания, дата последнего изменения.
  
![выбор_формы_в_реестре.png](/forms/выбор_формы_в_реестре.png)
  
В реестре вы можете:
- искать по номерам, названиям форм, датам, ответственным
- сортировать по номеру, дате создания, дате изменения, типу формы
- открывать форму для просмотра или редактирования (если она в черновике и у вас есть права)
- переходить в связанные формы, указанные в качестве значений полей, и возвращаться обратно
  
## Создание новой формы

В нужном реестре нажмите **+ Добавить.** Если у вас есть права на создание более чем одного дубликата данного типа формы, то выберите один из них, ориентируясь на суффиксы.
  
![выбор_формы_проекта.png](/forms/выбор_формы_проекта.png)

Система автоматически присвоит номер (следующий по порядку для данного типа формы проекта).

Форма создаётся в статусе «Черновик». В черновике можно сохранять любые значения, даже если обязательные поля не заполнены.

## Заполнение полей

При заполнении вы видите только те поля, которые не скрыты в настройках проекта и для которых выполнены условия видимости.
  
![карточка_формы.png](/forms/карточка_формы.png)

Типы полей отображаются соответствующими элементами управления:
- текстовое поле с возможной маской (например, для телефона)
- поле ввода числа с указанием единицы измерения и контролем допустимого диапазона/шага
- переключатель (да/нет)
- календарь, в котором может быть только дата или дата + время
- два календаря (начало, конец)
- список вариантов с одиночным или множественным выбором
- загрузка одного или нескольких файлов
- выбор одной или нескольких заполненных форм других типов (например, выбрать форму `Документ, подтверждающий качество` в качестве значения формы `Запись журнала входного контроля материалов`). 

Если в настройках поля с выбором заполненных форм используются фильтры, то список доступных для выбора форм автоматически ограничивается. Например, поле `Примененные материалы` может показывать только материалы, у которых `Тип материала` совпадает с выбранным в текущей форме типом.

## Завершение формы
  
Когда все обязательные поля заполнены, вы можете перевести форму в статус «Завершена». После этого форму нельзя редактировать и можно использовать в качестве значения полей других форм.
  
![завершение_формы.png](/forms/завершение_формы.png)

## Удаление и восстановление формы
  
Формы могут удалять и восстанавливать из корзины только редакторы и администраторы проекта. Восстановление возвращает форму в статус, который был до удаления.
  
![удаление_форм.png](/forms/удаление_форм.png)
