**:fire: Плиточный режим просмотра документов**

Реализовали плиточный режим просмотра в документах. Превью создается для  PDF, CAD моделей, изображений и видеофайлов при первом открытии документа любым из пользователей. У некоторых легких файлов до определенного размера превью будет создаваться при загрузке их в DOCS.
![плиточное_отображение_документов.png](/плиточное_отображение_документов.png)

**:fire: Функция печати из PDF**

Добавили функцию печати при просмотре PDF, которая позволяет открыть файл в просмотрщике вашего браузера для последующей печати.

![функция_печати_из_pdf.png](/функция_печати_из_pdf.png)

**:sparkles: Доработки функционала**
- Добавили отображение количества файлов и папок: общее и выбранных.
- Улучшили создание текстовых пометок в замечаниях:
	- при создании сразу включается режим редактирования (и без TYPE TEXT)
	- добавили настройку стилей и размеров текста
	- устранили ошибку со скроллом в процессе переноса на новую строку
	- добавили подсветку поля прозрачным цветом для пустой пометки
	- добавили выход из редактирования по Esc
	- добавили выделение всего текста при переход в редактирование (двойной клик)
- Добавили индексацию пометок в рамках одного замечания (чтобы в комментарии к замечанию можно было сослаться на конкретную пометку).
- Добавили удаление пометок кнопкой Delete и сброс выделения пометки кнопкой Esc.
- Добавили предупреждение о сохранении пометок при переключении между файлами.
- Доработали дисклеймер при простом просмотре: сместили вбок и убрали возможность его закрыть.

**:hammer_and_wrench: Исправление ошибок**
- Поправили отображение лупы при поиске в Firefox.
