<span style="background: linear-gradient(45deg, #00D1FF, #0695D7); color: white; padding: 2px 6px; border-radius: 4px; font-size: 24px;">DOCS Disk</span>

Опубликовали новую версию [DOCS Disk](/docs/disk): **2025.0.15**

**:sparkles: Доработки функционала**
- Оптимизировали работу с сетью.
- Доработали синхронизацию единичных файлов.
- Реализовали создание пустых папок из DOCS на Disc.
- Добавили поиск по текущей папке.
- Доработали систему логирования.
- Изменили индикацию для синхронизированных файлов, которые еще не скачали (был желтый треугольник, вернул облачко).
- Добавили множественную синхронизацию файлов (как для папок).
- Добавили версию приложения в заголовке основного окна.
  
**:hammer_and_wrench: Исправление ошибок**
- Исправили индикацию несоответствия версий в DOCS и на Disk (появился новый файл в доксе).
- Исправили ошибку с обновлением токена после длительного простоя приложения.
- Исправили ошибку с бесконечным лоадером при запуске приложения.
- Исправили ошибку с зависанием обработки файлов после отмены.
- Исправили ошибку с повторной синхронизацией файлов.

<span style="background: linear-gradient(45deg, #A09EFF, #4B47FF); color: white; padding: 2px 6px; border-radius: 4px; font-size: 24px;">DASHBOARD</span>

**:fire: Доступы в DASHBOARD по ролям**

Добавили возможность настривать доступы по ролям во вкладках [Управление](/dash/settings) и [Онлайн](/dash/intro).

Доступы назначаются Администратором проекта на трех уровнях: Отчет, Группа и Карточка (см. скриншот ниже). Права наследуются от высшего уровня к низшему, и их можно только усилить.

Доступно три вида прав:

> **Просмотр** — позволяет пользователю просматривать карточки во вкладках **Отчет** и **Управление.**
{.is-info}

> **Заполнение** — позволяет пользователю просматривать и редактировать данные внутри карточек во вкладке **Управление.**
{.is-success}

> **Редактирование** — позволяет пользователю перемещать, удалять, скрывать и переименовывать карточки.
{.is-warning}
