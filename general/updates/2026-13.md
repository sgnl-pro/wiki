**:sparkles: Доработки функционала**
- В окно загрузки файлов, которое появляется при наличии у папки масок, добавили поле с получателями уведомления о загрузке, а также поле для ввода комментария к версии файла.
- Сняли ограничение с кнопки `Обновить версии` при выделении файлов в комплекте согласования: теперь все выделенные неактуальные файлы будут обновлены, а актуальные пропущены.
- В настройках типа передачи добавили ввод значений свойств по умолчанию, которые будут подставляться при создании пакета (список оповещаемых) и формировании акта (отправитель, получатель, номер договора, название проекта).
- Добавили уведомление и лог при открытии комплекта ответственным за связанные замечания. Уведомление приходит администраторам проекта и пользователям, указанным на этапе `Оповещения` в настройках типа согласования. По умолчанию этот тип уведомления выключен в профиле пользователя.

**:hammer_and_wrench: Исправление ошибок**
- Исправили сжатие изображения по высоте при вставке факсимиле.
- Поправили отображение компании в списке проектов в мобильной версии приложения.
