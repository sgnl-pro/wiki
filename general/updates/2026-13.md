<span style="background: linear-gradient(45deg, #00D1FF, #0695D7); color: white; padding: 2px 6px; border-radius: 4px; font-size: 24px;">DOCS</span>

**:sparkles: Доработки функционала**
- В окно загрузки файлов (которое появляется при наличии у папки масок) добавили поле с получателями уведомления о загрузке и поле для ввода комментария к версии файла. Значения можно заполнять сразу для нескольких выбранных файлов. Также вывели в список файлов информацию о заполнении этих полей и обновляемые версии существующих в папке файлов:
![2026-13_-_окно_загрузки.png](/release_notes/2026-13_-_окно_загрузки.png =1200x)
- Сняли ограничение с кнопки `Обновить версии` при выделении файлов в комплекте согласования: теперь все выделенные неактуальные файлы будут обновлены, а актуальные пропущены.
- В настройках типа передачи добавили ввод значений свойств по умолчанию, которые будут подставляться при создании пакета (список оповещаемых) и формировании акта (отправитель, получатель, номер договора, название проекта).
- Добавили уведомление и лог при открытии комплекта ответственным за связанные замечания. Уведомление приходит администраторам проекта и пользователям, указанным на этапе `Оповещения` в настройках типа согласования. По умолчанию этот тип уведомления выключен в профиле пользователя.

**:hammer_and_wrench: Исправление ошибок**
- Исправили сжатие изображения по высоте при вставке факсимиле.
- Поправили отображение компании в списке проектов в мобильной версии приложения.

---
<span style="background: linear-gradient(45deg, #FF5867, #88383F); color: white; padding: 2px 6px; border-radius: 4px; font-size: 24px;">FORMS</span>

**:sparkles: Доработки функционала**
- Сняли ограничение с администратора проекта на редактирование завершенных форм.
- Добавили галочку "Уникальное значение в рамках проекта" для всех типов полей в конструторе форм. При активации пользователи не смогут сохранить форму со значением данного поля, если это значение уже используется в рамках данного типа формы проекта.
- Добавили возможность отображать значение поля типа "Список вариантов" в имени формы (например для отображения типа и единиц измерения при выборе материала или вида работ).
- Сняли ограничение с количества вариантов у поля типа "Список вариантов": теперь можно оставить один вариант (например для обязательного подтверждения согласия в некоторых формах).
- Добавили предупреждение при закрытии формы, в которую внесли изменение и не сохранили.
- Добавили открытие форм из полей других форм в таблице.
- Добавили корзину для восстановления удаленных типов форм в настройках проекта.
- Разрешили использовать цифры в системных именах полей (системные имена используются в шаблонах для генерации DOCX).
- Добавили логи изменений форм в организации HUB.

**:hammer_and_wrench: Исправление ошибок**
- Поправили сортировку отображаемых полей в имени формы.

**:fire: Карточка 18_Таблица из DOCS**

Новая карточка 18_Таблица из DOCS предназначена для отображения данных из Excel-файлов, загруженных в DOCS. При настройке карточки можно выбрать лист книги, указать диапазон отображаемых ячеек и настроить относительную ширину столбцов для оптимального отображения таблицы на превью карточки.

![2026-13_-_карточка_18.png](/release_notes/2026-13_-_карточка_18.png =1200x)

![2026-13_-_карточка_18_-_2.png](/release_notes/2026-13_-_карточка_18_-_2.png =1200x)

**:sparkles: Доработки функционала**
- Добавили стрелочки для переключения между карточками в группе.
- Убрали обозначения 3D карточек при экспорте, а при про просмотре теперь они отображаются при наведении.
- В карточки 50 и 51 добавили предупреждение о разных единицах измерения.

**:hammer_and_wrench: Исправление ошибок**
- Исправили ошибку, при которой в карточке 15 не выводилась разница между последней и предполследней неделями.
- Исправили ошибку при открытии карточки 41.
- В карточках 41, 41 MPP и 46 добавили перенос текста на превью.
- Поправили выгрузку карточек 41, 41 MPP, 46, 61 и 62, чтобы отображались актуальные даты (со скроллом вправо), а не начальные.
- В карточках 50 и 51 исправили скрытие легенды.
- При экспорте карточки 90 добавили сообщение с текстом о скрытии конфиденциальной информации.
