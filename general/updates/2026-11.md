<span style="background: linear-gradient(45deg, #2D4E67, #112538); color: white; padding: 2px 6px; border-radius: 4px; font-size: 24px;">HUB</span>

:sparkles: Добавили в разделе "Проекты" системную роль "Как в организации". Эта роль отвечает за синхронизацию ролей, назначенных во вкладке "Пользователи", во вкладку "Проекты". При добавлении пользователя в проект по умолчанию назначается роль "Как в организации". Удобно, если не требуется индивидуально настраивать роли в каждом проекте.

![синхронизация_ролей_как_в_организации.png](/синхронизация_ролей_как_в_организации.png)

**:hammer_and_wrench: Исправление ошибок**
- Поправили выбор локации в настройках проекта

<span style="background: linear-gradient(45deg, #00D1FF, #0695D7); color: white; padding: 2px 6px; border-radius: 4px; font-size: 24px;">DOCS</span>

**:fire: Массовое редактирование и удаление типов согласования**

Реализовали возможность массового редактирования и удаления типов согласования в настройках. Поля с разными значениями помечаются соответствующей надписью.

![массовое_радктирование_типов_согласования.png](/массовое_радктирование_типов_согласования.png)
  

**:sparkles: Доработки функционала**
- Оптимизаровали работу с фильтрами в разделе "Документы": 
добавили ленивую пагинацию для отображения файлов, что позволило устранить проблему с отображением только первых 100 файлов при применении фильтров в разделе "Документы".
- Доработали копирование и перемещение файлов в папки с масками.
- Разрешили использовать символы [<>:"\/|?*] в шифре передачи.
- Добавили подсвечивание файла в папке при его закрытии.
- Убрали панель навигации в файле при открытом замечании.
- Привели интерфейс настроек колонок в соответствии с дизайном.

**:hammer_and_wrench: Исправление ошибок**
- Исправили загрузку файлов с скрытыми символами.
- Устранили проблемы с ломающимся интерфейсом при заполнении в замечании пометки "Текст".
- Устранили ошибку при назначении атрибута через контекстное меню.

<span style="background: linear-gradient(45deg, #A09EFF, #4B47FF); color: white; padding: 2px 6px; border-radius: 4px; font-size: 24px;">DASHBOARD</span>

**:fire: Расширение функционала легенды для 3D карточек**
Добавили во все 3D карточки возможность выбрать цвета из TOOLS для настройки Легенды. 
Также теперь цвета легенды стали кликабельны и изолируют выборки по свойству "UserColor".
![легенда_в_3д_карточках.jpg](/легенда_в_3д_карточках.jpg)

**:sparkles: Доработки функционала**
- В [карточке 22](/dash/cards-20#card-22) добавили возможность выбрать Тип документа при импорте данных из 1С; добавили возможность вручную добавить строки при уже вставленных данных из 1С; убрали сбрасывание столбца ПЛАН при импорте данных.
- В [карточке 15](/dash/cards-10#card-15) добавили возможность указывать пустое значение в таблице.

**:hammer_and_wrench: Исправление ошибок**
- В карточке 64 исправили расчет данных.
- Поправили отображение карточке 33 и 22 в разделе Отчет.
- Исправили обновление превью карточки 67.
