**:fire: Конвертер Tangl для Revit файлов**

> Функционал доступен в бета-режиме: иногда может не срабатывать. Планируем стабилизировать его в следующих версиях.
{.is-warning}

Вместе с партнёрами из Tangl реализовали открытие загруженных файлов RVT через [Tangl Viewer](/docs/viewers) без дополнительных действий. Ранее для открытия Revit моделей в этом просмотрщике требовалась выгрузка через специальный плагин.

![конвертация_rvt_в_tangl.png](/конвертация_rvt_в_tangl.png)

**:fire: Сообщение в режиме простого просмотра**

Реализовали вывод информационного сообщения (например дисклеймера для покупателей квартир) при открытии опубликованной модели в режиме [простого просмотра](/docs/sharedSettings). Наличие и текст сообщения можно задать в [Сервисы](/docs/settings#services).

![надпись_в_простом_просмотре.png](/надпись_в_простом_просмотре.png)

**:sparkles: Доработки функционала**
- Добавили кнопку перехода в [комплект согласования](/docs/reviews) после создания его из [документов](/docs/folders).
- Добавили автоматическое напоминание по этапу согласования в последний день этапа.
- Добавили быстрые фильтры по типу передачи в [реестре передач](/docs/documentsTransfers).

**:hammer_and_wrench: Исправление ошибок**
- Исправили отображение отклонённых комплектов согласования в статусе **Черновик** при фильтрации.
- Исправили значение дельты завершённых согласований при экспорте.
- Исправили ошибку, при которой файл SIG отображался со штампом в комплекте согласования.
- Исправили ошибку, при которой при добавлении файлов в комплект согласования отображались не все комплекты.
