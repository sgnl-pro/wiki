<span style="background: linear-gradient(45deg, #00D1FF, #0695D7); color: white; padding: 2px 6px; border-radius: 4px; font-size: 24px;">DOCS</span>

**:sparkles: Доработки функционала**
- Добавили оповещение всех участников при завершении [согласования](/docs/reviews).
- В [пакетах передачи](/docs/documentsTransfers) добавили просмотр файлов PDF для всех, у кого есть доступ к пакету.
- В настройках проекта во вкладке **[Сервисы](/docs/settings#services)** добавилили возможность для каждого клиента Tangl указывать свои адреса серверов (если Tangl on-premise).
- В [публичный API](/general/api) добавили метод для получения событий по всем замечаниям проекта.

**:hammer_and_wrench: Исправление ошибок**
- Исправили ошибку при возвращении в реестр согласований из ранее отфильтрованного комплекта.
