**:fire: Работа с согласованием прямо из файла**

В [верхней панели файла](/docs/viewers) теперь отображается статус [согласования](/docs/reviews) — как при просмотре в папке.

При наличии прав на просмотр согласования можно нажать на статус и открыть поверх документа мини-окно с отображением цепочки согласования, выделенным текущим этапом, а также номером, шифром и названием комплекта. Далее расположен статус открытого файла, который можно менять прямо здесь (при наличии прав на согласование текущего этапа). Далее расположен список других файлов комплекта, между которыми можно переключаться. При пространовке статусов у всех файлов комплекта доступна кнопка **Завершить согласование.**

Статус файла и мини-окно **Согласование** доступно при открытии файла как из папки, так и из комплекта согласования. Таким образом, согласование комплекта теперь можно проводить непосредственно при просмотре файлов, не закрывая их каждый раз:

![2025.38_-_согласование_из_файла.png](/release_notes/2025.38_-_согласование_из_файла.png)

**:fire: Группы согласований**

Типам согласований теперь можно назначать группы для фильтрации. Группы создаются и редактируются непосредственно в [настройках типа](/docs/settings#reviews):

![2025.38_-_настройка_групп_согласований.png](/release_notes/2025.38_-_настройка_групп_согласований.png)

В реестр согласований добавили соответствующий фильтр:

![2025.38_-_фильтр_групп_согласований.png](/release_notes/2025.38_-_фильтр_групп_согласований.png =600x)

**:fire: Публичная ссылка на пакет передачи**

Реализовали возможность публиковать акт [передачи](/docs/documentsTransfers) для подписания сторонним лицом, не имеющим доступа к системе или конркетному проекту. Публикация настраивается на уровне [типа передачи](/docs/settings#documentsTransfers): можно сразу задать Email для всех передач данного типа, а можно оставить поле пустым — в этом случае ввести получателя нужно будет при отправке конкретной передачи.

Внешнему пользователю придёт письмо с публичной ссылкой, по которой он сможет перейти, ввести имя и подписать акт приема-передачи и скачать файлы пакета.

![2025.38_-_публичный_пакеты_передач.png](/release_notes/2025.38_-_публичный_пакеты_передач.png)

**:fire: Создание графиков Ганта**

> Функционал доступен в бета-режиме.
{.is-warning}

Реализовали функционал создания и редактирования графиков Ганта в DOCS в собственном расширении `.sgmpp`, который в дальнейшем планируем развивать в отдельном модуле для постановки задач:

![2025.38_-_создание_графика.png](/release_notes/2025.38_-_создание_графика.png =600x)

Сейчас редактор позволяет вести дерево задач с их автоматической нумерацией, задавать даты начала и конца, прогресс в процентах, добавлять пользовательские колонки. На графике можно перемещать задачи, назначать и менять их сроки, ставить зависимости между задачами. При просмотре можно менять масштаб графика:

![2025.38_-_график_гантта.png](/release_notes/2025.38_-_график_гантта.png)

**:sparkles: Доработки функционала**
- Добавили копирование значений атрибутов файлов и коментариев версий при копировании через согласование. Атрибуты копируются только в случае создания первой версии нового файла (существующие атрибуты не перетираются).
- Добавили в [публичный API](/general/api) методы для работы с вложениями к замечаниям `/public/v1/issues/attachments` и `/public/v1/issues/attachments/object/download`.

**:hammer_and_wrench: Исправление ошибок**
- Убрали окно выбора просмотрщика моделей Tangl или Forge в случае, если в настройках проекта включен только один из них. Теперь модели открываются сразу в нужном просмотрщике.
- Поправили, чтобы для просмотра актуального статуса обработки модели не приходилось обновлять страницу.
- Исправили некоторые другие ошибки при взаимодействии с моделями.
- Исправили ошибку, при которой слетали галочки с выбранных файлов при копировании через панель инструментов и открытии окна **Куда скопировать.**
- Убрали предупреждение **Замечание не сохранено.**
- Убрали ссылки из окна прикрепления файла к замечанию.
- Убрали лишние поля при отклонении согласования на финальном этапе: выбор папки, чек-бокс о завершении с замечаниями и тип штампования.
- Поправили, чтобы при переключении типа согласования в настройках курсор ставился на поиске.
