**:fire: Линейка для измерения PDF**

Реализовали инструмент измерения расстояния между двумя точками при [просмотре PDF](/docs/viewers#pdf):

![линейка.jpg](/release_notes/линейка.jpg =1200x)

При первом измерении документа линейку нужно откалибровать, указав длину известного отрезка. Калибровка сохраняется для всех пользователей. Также можно откалибровать отдельный лист, если его масштаб отличается от остальных:

![калибровка.jpg](/release_notes/калибровка.jpg =1200x)

**:sparkles: Доработки функционала**
- Оптимизировали открытие больших PDF с помощью серверной синхронизации, в том числе на iOS. В мобильной версии стали доступны кнопки синхронизации и отмены загрузки документа.
- Добавили дополнительный прокси для [Forge](/docs/viewers#bim) (три точки в правом верхнем углу). Включайте прокси, если испытываете проблемы с открытием моделей:
![прокси.png](/release_notes/прокси.png =600x)
- На верхней панели файла вывели путь до корневой папки. Аналогично сделали при просмотре папки: теперь при нажатии на три точки открывается весь путь, любую папку можно открыть в новой вкладке колёсиком мыши или с сажатым <kbd>Ctrl</kbd>:
![хлебные_крошки.png](/release_notes/хлебные_крошки.png =600x)
- Добавили возможность скачать вложения из картчоки [замечания](/docs/issues) архивом:
![скачать_архив_вложений.png](/release_notes/скачать_архив_вложений.png =600x)
- Реализовали сохранение значений одинаковых атрибутов при смене типов замечаний.
- Добавили отображение удаленных из [согласования](/docs/reviews) версий файлов:
![просмотр_удаленных_файлов_в_согласовании.png](/просмотр_удаленных_файлов_в_согласовании.png =600x)
- Добавили версии файлов с датами их создания в акт [передачи](/docs/documentsTransfers#downloading):
![версии_файлов_в_акте.png](/release_notes/версии_файлов_в_акте.png =600x)

**:hammer_and_wrench: Исправление ошибок**
- Добавили игнорирование лишних пробелов при поиске.
- Закрепили названия столбцов в [документах](/docs/folders).
- Вернули столбец `Путь` при фильтрации файлов влево, после `Названия`.
- Исправили ошибку при изменении настроек отображения в сравнении PDF.
- Поправли закрытие офисных документов в мобильной версии.
- Добавили предупреждение об отсутствии активных ролей при запуске [согласования](/docs/reviews) из документов.
- Исправили некорректное отображение плановой и договорной даты в календаре согласований в некоторых случаях.
- Поправили сортировку в окне выбора файлов в согласовании.
- Исправили ошибку при перезапуске согласования у пользователей с правами на создание в случае, когда согласование было запущено администратором проекта.
- Отключили копирование списка пользователей или ролей при нажатии на этап согласования.
- Убрали закрытия окна после добавления файлов в [пакет передачи](/docs/documentsTransfers#creating) из документов, чтобы можно было перейти в пакет.
- Из этого же окна убрали уже запущенные пакеты передачи.
- Поправили иконку профиля в мобильной версии.
