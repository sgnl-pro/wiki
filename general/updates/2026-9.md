**:fire: Настройки уведомлений в профиле пользователя**

В разделе [Уведомления в профиле HUB](/hub/user#notifications) появились расширенные настройки. Как и ранее, можно выбрать, куда вы хотите их получать (сейчас временно доступна только почта). Также в общих настройках можно настроить объединение уведомлений в одно письмо, произошедших за выбранный период. Если объединение выключено, то каждое уведомление будет приходить отдельно:

![2026-9-настройки_уведомлений-общие.png](/release_notes/2026-9-настройки_уведомлений-общие.png =600x)

Далее по модулям можно включить / отключить и настроить объединение для уведомлений каждого типа, которые сейчас доступны в системе. Этот список будет расширяться:

![2026-9-настройки_уведомлений-docs.png](/release_notes/2026-9-настройки_уведомлений-docs.png =1200x)

В настройках DOCS осталась возможность выключить уведомления для конкретного проекта и управлять списком папок, по которым вы хотите получать уведомления:

![2026-9-настройки_уведомлений-проект.png](/release_notes/2026-9-настройки_уведомлений-проект.png =900x)

Подробнее о настройках уведомлений см. в [статье](/hub/user#notifications).
