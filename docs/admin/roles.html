В сервисе SIGNAL DOCS «Роли» служат для возможности настраивать доступ участников проекта в зависимости от их обязанностей к дополнительным модулям и элементам системы.

 

1. Для того чтобы создать роль перейдите на сайт hub.sgnl.pro → Компания → вкладка Роли и нажмите "Добавить"

 

 

2. В появившемся окне задайте название роли и нажмите "Добавить"

 

 

3. После создания всех необходимых ролей перейдите на вкладку ПОЛЬЗОВАТЕЛИ, нажмите на три точки в столбце “Роль” и назначьте для пользователя соответствующую роль. Теперь пользователю автоматически будет назначаться роль при добавлении его в новый проект. Для разных пользователей можно назначить одну и ту же роль, например, Проектировщики.

 

 

Чтобы поменять роль пользователю в уже существующем проекте перейдите во вкладку ПРОЕКТЫ, нажмите на три точки в столбце “Роль” и назначьте для пользователя соответствующую роль на данном проекте.

 

 

4. После этого Роли станут доступны в настройках прав доступа в SIGNAL DOCS
