Функция календарь позволяет указывать рабочие и выходные дни, принятые в вашей организации. календарь влияет на график согласований по проекту. Количество дней на каждом этапе согласования вычисляется с учетом выходных дней, указанных в календаре.

 

Для создания календаря перейдите во вкладку "Календарь"

 

 

 

В открывшемся окне введите название календаря, год и откорректируете выходные/рабочие дни. Рабочими считаются дни в синих кружках. Нажмите кнопку сохранить

 

 

Вернитесь в список доступных календарей и активируйте необходимый для работы календарь. Выбранный календарь будет предлагаться по умолчанию при создании новых проектов.
